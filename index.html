PROJECT_01 · MSc DISSERTATION GITHUB

Network Intrusion Detection System (IDS)

MSc dissertation at University of Essex — built a full end-to-end, leakage-safe ML pipeline for flow-based network intrusion detection, comparing classical models (Random Forest, SVM, k-NN) against deep architectures (MLP, CNN, LSTM) across 11 attack classes on ~300,000 flow records.

  • Engineered a strict split-then-fit preprocessing pipeline (median imputation, IQR outlier removal, variance filtering) to eliminate data leakage — a critical flaw in most published IDS studies
  • Compared 3 feature selection families — Mutual Information (filter), RF Importance (embedded), RFE (wrapper) — using Jaccard overlap and cross-seed stability analysis; identified a compact core of 5–10 features achieving ≥95% of peak macro-F1
  • Evaluated beyond accuracy: reported ROC-AUC, PR-AUC, Brier score, and ECE calibration metrics; applied Platt and isotonic post-hoc calibration to improve probability reliability without loss of ranking performance
  • Ran Wilcoxon signed-rank hypothesis tests across 3 random seeds; confirmed H4 — ≤10 features achieves ≥95% peak F1 — providing a clear performance–latency trade-off for deployment
  • Random Forest + RFIMP (K=5) achieved macro-F1 = 1.000 with 0.43s training time; SVM + MI (K=10) achieved F1 = 0.999 at sub-second inference — both recommended for real-world SOC deployment
