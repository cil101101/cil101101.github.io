PROJECT_01 · MSc DISSERTATION GITHUB

Network Intrusion Detection System (IDS)網路入侵偵測系統（IDS）

MSc dissertation at University of Essex — built a full end-to-end, leakage-safe ML pipeline for flow-based network intrusion detection, comparing classical models (Random Forest, SVM, k-NN) against deep architectures (MLP, CNN, LSTM) across 11 attack classes on ~300,000 flow records.

Essex 大學碩士論文——建立完整端對端、防洩漏 ML pipeline，比較傳統模型與深度架構在約 30 萬筆流量記錄及 11 種攻擊類別上的表現。

  • Engineered a strict split-then-fit preprocessing pipeline to eliminate data leakage — a critical flaw in most published IDS studies
  • 建立嚴格的先分割後擬合預處理 pipeline，消除資料洩漏——多數已發表 IDS 研究的關鍵缺陷
  • Compared 3 feature selection families using Jaccard overlap and cross-seed stability; identified 5–10 feature core achieving ≥95% peak macro-F1
  • 比較 3 種特徵選取方法，找出 5–10 個特徵核心達到 ≥95% 峰值 macro-F1
  • Evaluated ROC-AUC, PR-AUC, Brier score, ECE; applied Platt and isotonic calibration for improved probability reliability
  • 評估 ROC-AUC、PR-AUC、Brier 分數和 ECE；套用校準方法提升機率可靠性
  • Wilcoxon signed-rank tests confirmed H4: ≤10 features achieves ≥95% peak F1 — clear performance–latency trade-off for SOC deployment
  • Wilcoxon 符號秩檢定確認 H4：≤10 個特徵可達 ≥95% 峰值 F1，為 SOC 部署提供清晰的效能–延遲權衡
  • Best result: RF + RFIMP (K=5) macro-F1 = 1.000 at 0.43s training; SVM + MI (K=10) F1 = 0.999 sub-second inference
  • 最佳結果：RF + RFIMP（K=5）macro-F1 = 1.000，訓練時間 0.43 秒；SVM + MI（K=10）F1 = 0.999
